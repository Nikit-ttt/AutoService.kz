# AutoService — Онлайн-запись в автосервис
Удобный веб-сайт, который помогает автовладельцам быстро находить нужные услуги и записываться на ремонт, а администраторам — управлять заявками и базой данных через удобную панель.

## Что умеет этот проект
* **Главная страница:** Красивое приветствие, информация о сервисе и актуальные акции.
* **Каталог услуг:** Список всех работ (замена масла, шиномонтаж и др.) с ценами, временем выполнения, а также поиском и сортировкой.
* **Регистрация и Вход:** Полноценная система авторизации пользователей. Данные надежно сохраняются в базе данных, а сессия запоминается сайтом.
* **Личный кабинет администратора:** Специальная панель (`Admin.html`), где админ может управлять услугами, пользователями и статусами заявок.
* **Переключение тем:** Поддержка темного и светлого режимов оформления.

## Как это устроено (Структура проекта)
Проект разделен на две основные части:
1. **Frontend: Чистый HTML, CSS-стили для красивого отображения и JavaScript для работы интерактивных элементов, отправки форм и запросов к серверу.
2. **Backend (Серверная часть): Сервер на платформе Node.js (Express), который принимает запросы с сайта, обрабатывает их и связывается с базой данных SQLite.

## Что нужно для работы (Зависимости)
Проект написан на JavaScript и работает с помощью:
* **Node.js** (платформа для запуска серверного кода)
* **Express** (библиотека для быстрой настройки сервера и маршрутов)
* **SQLite3** (легкая и быстрая база данных, которая хранится в одном файле `project.db`)
